# 入力形式
H W
c[1][1]c[1][2]...c[1][W]
c[2][1]c[2][2]...c[2][W]
 :
c[H][1]c[H][2]...c[H][W]

# 出力形式
鉄道路線が通るマスの数の最大値を出力してください。
条件を満たす鉄道路線がない場合は、代わりに -1 を出力してください。

----------

# 入力例 1
3 3
...
.#.
...

# 出力例 1
8

----------

# 入力例 2
1 6
......

# 出力例 2
-1

----------

# 入力例 3
4 4
....
#...
....
...#

# 出力例 3
12
